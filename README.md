## vehicle-object-into-Json
## Sample code to check the vehicle object into Json
```sh
class Vehicle:
    def __init__(self, name, engine, price):
        self.name = name
        self.engine = engine
        self.price = price
```
## Expected Output
{
    "name": "Toyota Rav4",
    "engine": "2.5L",
    "price": 32000
}
